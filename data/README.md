# Data Folder

This project uses large climate datasets that are not included in this repository.

Datasets used:

CMIP6 Models:
- CESM2
- IPSL-CM6A-LR
- MPI-ESM1-2-LR

CERES Satellite Observations.

You can download the datasets from:

CMIP6:
https://esgf-node.llnl.gov/projects/cmip6/

CERES:
https://ceres.larc.nasa.gov/data/
